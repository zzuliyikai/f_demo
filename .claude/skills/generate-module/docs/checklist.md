## 常见问题排查

### 编译错误

**错误**: `Target of URI doesn't exist`
- [ ] 检查文件路径是否正确
- [ ] 检查文件名是否匹配
- [ ] 运行 `flutter pub get`

**错误**: `Could not find the correct Provider`
- [ ] 检查 Repository 是否已注册
- [ ] 检查 Provider 类型是否匹配
- [ ] 检查导入是否正确

### 运行时错误

**错误**: `Route not found`
- [ ] 检查路由是否已注册
- [ ] 检查路由名称是否正确
- [ ] 检查是否使用 `routePage()`

**错误**: `Null check operator used on a null value`
- [ ] 检查数据是否为空
- [ ] 添加空值处理
- [ ] 使用默认值

### 功能问题

**问题**: Mock 模式不生效
- [ ] 完全重启应用
- [ ] 检查 `$appCache.mockRepo` 的值
- [ ] 检查 `_getRepo()` 调用

**问题**: 数据不更新
- [ ] 检查是否调用了 `emit()`
- [ ] 检查是否使用了 `nextState()`
- [ ] 检查 BlocBuilder 是否正确配置
